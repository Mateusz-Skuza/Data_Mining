# Data Mining

This repository contains projects focused on supervised learning and practical classification techniques. The analyses use classic datasets, such as `iris` and `wine`, to demonstrate how different models handle data structures.

## Report 1
This project demonstrates how to use linear regression for classification tasks. Key topics include:
* **Linear Indicator Matrix**: Implementing a classifier by building separate regression models for each class.
* **Feature Expansion**: Improving model accuracy by adding second-order polynomials and interactions between variables.
* **Model Diagnostics**: Analyzing the "masking effect" and evaluating performance through confusion matrices and decision boundary plots.

## Report 2
This report focuses on LDA as a method for both classification and reducing the number of variables in a dataset. The analysis includes:
* **Dimensionality Reduction**: Transforming high-dimensional data into a simpler form while keeping classes as separate as possible.
* **Boundary Analysis**: Finding the most efficient linear boundaries to distinguish between multiple groups.
* **Error Evaluation**: Comparing how well the model performs across different sets of features.
